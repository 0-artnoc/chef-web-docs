This subcommand has the following options:

``--api-port=PORT``
   |port delivery_api|

``-e=ENTERPRISE``, ``--ent=ENTERPRISE``
   |delivery enterprise|

``--raw``
   Print the raw token. 

``-s=SERVER``, ``--server=SERVER``
   |delivery server|

``-u=USER``, ``--user=USER``
   |delivery user|

``--verify``
   Verify if a token is a valid token.
