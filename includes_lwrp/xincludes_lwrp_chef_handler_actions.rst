This custom resource provider has the following actions:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Action
     - Description
   * - ``:enable``
     - Enable the handler for the current |chef client| run on the current node.
   * - ``:disable``
     - Disable the handler for the current |chef client| run on the current node.
