For a |chef server| configuration that is launched from the |amazon aws marketplace|, access to the instance on the following ports is required:

.. list-table::
   :widths: 60 420
   :header-rows: 1

   * - Port
     - Description
   * - 443
     - HTTPS for |chef manage|
   * - 8443
     - HTTPS for |chef analytics|
   * - 22
     - SSH
